Welcome to TensorVision
=======================

TensorVision is a library to build, train and evaluate neural networks in
TensorFlow.

TensorVision is a work in progress, input is welcome. The available
documentation is limited for now. The project is on `GitHub`_.

User Guide
------------

The TensorVision user guide explains how to install TensorVision, how to build
